# Extreme Wordle

A Wordle clone made with Svelte and TailwindCSS.

To take a spin on the game, I decided to add words of varying lengths.

Hosted on Vercel.com.

## Developing

```bash
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Building

```bash
npm run build
```
